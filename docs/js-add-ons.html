Take Five! – JavaScript Add-Ons

This page includes a few scripts that allow to

  • Clean the browser's history after leaving a presentation (js-add-ons/takefive-clean-history.js)
  • Leave a presentation by pressing Esc on the keyboard (js-add-ons/takefive-keyboard-events.js)
  • Swipe left and right via touch events (js-add-ons/takefive-touch-events.js)

A further script that allows to use an empty hash (#) as the exit link and leave no traces in the address bar is available (js-add-ons/takefive-clean-empty-hash.js), however this is alternative to takefive-clean-history.js and can be used only when the latter is not used.
